# Chat AI NVDA Add-on

**Chat AI** is an NVDA add-on that allows you to chat with AI models in a simple and accessible way.

This add-on is currently in early development and may change in future versions.

## Features

- Simple and accessible AI chat interface
- Session-based chat history
- Custom API key support
- Multi-turn conversation within a single session
- Keyboard shortcut for quick access

## Supported AI

- Gemini (Google AI)

More providers may be added in future updates.

## Usage

1. Open the chat dialog by pressing `NVDA + Alt + A`
2. Type your prompt in the input field
3. Send your message
4. Read the response in the chat history area

## Settings

You can open the add-on settings with:

`NVDA + Shift + Alt + A`

Current settings include:

- API provider selection
- API key entry

## Installation

1. Download the `.nvda-addon` file from the Releases page
2. Open the file
3. Confirm installation in NVDA
4. Restart NVDA if needed

## Notes

- This add-on is still under development
- Features and interface may change
- Internet access is required
- A valid API key is required for supported providers
